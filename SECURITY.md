# Security Policy

## Supported versions

Only the latest release is supported with security fixes.

## Reporting a vulnerability

If you discover a security issue, please **do not** open a public GitHub issue.
Instead, send an email or open a private advisory via
[GitHub Security Advisories](https://github.com/nickybulthuis/departic/security/advisories/new).

## Scope

Departic runs on your local network and talks to your EVCC instance. It does
not expose any authentication mechanism — it is designed to be deployed behind
a reverse proxy or on a trusted network.
